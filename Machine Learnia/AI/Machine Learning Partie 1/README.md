## üìÇ Files Description

###  Machine_Learning_Regression_Affine_Pro_Unidimensionnelle.py
Code based on the following video:  
https://youtu.be/vG6tDQc86Rs  

Professional implementation of **unidimensional linear regression**, introducing core machine learning concepts such as loss functions and optimization.

---

###  Machine_Learning_Regression_Affine_Unidimensionnelle.py
A minimal demonstration of **linear regression** using only **two data points**, designed to illustrate the fundamental principles of model fitting.

---

###  Machine_Learning_Regression_Affine_Dimension_2.py
An extension of `Machine_Learning_Regression_Affine_Pro_Unidimensionnelle.py` that supports **two input features**, introducing multivariate linear regression.

---

###  Machine_Learning_Regression_Polynomial_Quadratic_Unidimensionnelle.py
Code based on the following video:  
https://youtu.be/cpltYCNLIt0  

Implementation of **quadratic polynomial regression** (degree 2) for a single feature.

---

###  Machine_Learning_Regression_Polynomial_Cubic_Unidimensionnelle.py
An upgrade of the quadratic polynomial regression model, extending the equation to **degree 3** to better fit more complex datasets.

---

###  Machine_Learning_Regression_Polynomial_Quadratic_Dimension_2.py
An extension of the quadratic polynomial regression model to support **multiple input features**.

---

###  Machine_Learning_Regression_Polynomial_Degree_Unidimensionnelle.py
A dynamic polynomial regression model where the **degree is chosen by the user**, allowing experimentation with underfitting and overfitting.

---

###  Machine_Learning_Regression_MultiPolynomial_Dimension_2.py
A multivariate polynomial regression model that automatically **increases the polynomial degree** if the model‚Äôs accuracy is insufficient.

---

###  Machine_Learning_Regression_Trigonometric_Unidimensionnelle.py
A regression model based on **trigonometric features** (`sin` and `cos`), similar to a **Fourier-like transformation**, up to second degree.

---

###  Machine_Learning_Regression_Trigonometric_Degree_Unidimensionnelle.py
An extended version of the trigonometric regression model where the **degree is chosen by the user**, allowing flexible function approximation.
